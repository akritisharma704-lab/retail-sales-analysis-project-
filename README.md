Retail Sales Performance Analysis
📊 Project Overview

This project analyzes 150,000+ retail transaction records to evaluate revenue performance, discount impact, customer purchasing behavior, and store-level performance. The analysis was conducted using SQL for data exploration and Power BI for visualization and insight generation.

🛠 Tools & Technologies

- SQL (Data Analysis & Aggregation)

- Excel (Data Validation & Quick Analysis)

- Power BI (Dashboard & Visualization)

🎯 Business Objectives

- Analyze monthly revenue trends and growth patterns

- Evaluate the effectiveness of discount strategies

- Identify high-value and repeat customers

- Detect underperforming stores and regions

- Assess product category contribution to total revenue

📈 Key KPIs Tracked

- Total Net Revenue

- Gross Sales

- Average Order Value (AOV)

- Revenue by Category

- Revenue by Store

- Discount Impact on Revenue

🚀 Outcome

The project provides actionable business insights to optimize discount strategies, improve store performance, and enhance revenue growth.
